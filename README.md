# :speech_balloon: Node & Express Chat Application

Chat application built with Node, Express and Socket.io. (2018)


## :floppy_disk: Install

Dependencies:
* body-parser
* express
* socket.io

NPM:
```js
"scripts": {
    "dev": "node server.js"
 },
 ```

Run:
* npm run dev
